Stock Dashboard

Stock Dashboard is a personal local investment dashboard used only by the account owner.

The app helps the account owner track portfolio holdings, import depot trade confirmations, generate local decision-support summaries, and send personal action summaries.

The app uses Gmail readonly access only to read emails labelled depot-import for portfolio import. The app uses Gmail send access only to send action summaries to the account owner.

All processing is for personal use. Data is processed locally and is not sold, shared, or transferred to third parties.
